# IMFDatasetInfo (`src/pages/IMFDatasetInfo.tsx`)

## Purpose
- Static informational page describing the IMF datasets (WEO and NEA) referenced by the dashboard.
- Provides citation guidance and external resource links for researchers.

## Key Interactions
- Pure presentational component; no props, hooks, or client-side effects.

## Usage Notes
- Update dataset descriptions and citation years as IMF releases change; no runtime dependencies mean edits are straightforward content tweaks.
